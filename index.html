Introduction

Vitamin provides a simple and easy to use ActiveRecord implementation for working with your database. Each database table or view is wrapped into a "Model" class. Thus, a model instance is tied to a single row in the table. Models allow you to query for data in your tables, as well as inserting or updating records.

Defining Model classes

To get started, let's define a basic Model using the extend static method, and specify the primary key name, and data source name

var Model = require('vitamin/model')

var Post = Model.extend({
  /**
   * The table or the collection name associated with this model
   */
  getSourceName: function () { return "posts" },

  /**
   * The primary key name, by default 'id' is used
   */
  getKeyName: function () { return "id" },

})
